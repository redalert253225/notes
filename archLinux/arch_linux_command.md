$ lsblk 查看硬碟列表

$ sudo pacman -S package_name   安裝軟件

$ sudo pacman -R package_name   移除軟件

 $ sudo pacman -Syu  更新所有軟體包

$ netctl start your_WIF_file_name     開啟你設定的WIF

$ ip add  查看 interface

$ systemctl start dhcpcd@<interface>  開啟有限網路

$ rm -rf  /XXX/XX 刪除資料夾及其內容

$ mkdir /XX/XX   創建資料夾

$ touch /XX/XX.py  創建檔案

$ netstat -rn      查看 ip Gateway  Genmask  , 要安裝net-tools
